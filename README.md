MOAT js Testing Samples
========

これは、サーバープラグインのテストパターンをまとめるためのプロジェクトです。

## スクリプト

- `scenario01.js`

## コマンド

### Set up

    npm install

### Build & Unit Tests

    grunt

### Packaging

    grunt pack

### Cleaning

    grunt clean

### Test

    grunt nodeunit
